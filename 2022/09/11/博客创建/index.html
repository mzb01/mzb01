博客创建

下载node和git，这个应该都没什么问题，不会的可以搜一下

难得就是下载之后如何创建

在这里附上链接：https://www.bilibili.com/video/BV1A4411G7SF?p=1

讲讲跟着操作所犯下的错误

在GitHub上创建库的时候

记得库名一定是<名字>.github.io

不然等你一切创建好了之后，将文章，背景等等部署上的时候，会发现没有变化

其次讲讲….‘npm fund’…问题，这个问题我也搜了很多，得到的办法

这个不是报错，这个只是一些捐资，如果不想看到，在执行的命令后面加上–no-fund即可
fatal: Not a git repository (or any of the parent directories): .git

这个问题，想必很多人遇到过,我的是在blog文件夹里面使用git bash here然后输入hexo init出现的这个问题，解决办法还是将该文件夹里面的所有东西都删除

遇到过很多问题，在博客创建成功之后几天才写的博客，很多问题都忘了，基本上可以说能遇到的问题我都遇到了，像什么下载东西下不上，就把可牛文件夹删了，结果好了。如有什么问题解决不了的，可以评论
